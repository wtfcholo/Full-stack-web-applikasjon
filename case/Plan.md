Konsept:
Et lite bakeri som ønsker en nettside hvor kunder kan- se informasjon om bakeriet, sjekke åpningstider og kontaktinfo, Se meny med tilgjengelige bakevarer, legge inn bestilling gjennom et bestillingsskjema.

Modul 1:
Opprette database med tabeller for produkter, bestillinger og bestillingerslinjer, Er-diagram, SQL-fil for å lage tabellene, dokumentere spørringer for å see inn og hente ut bestillinger.

Modul 2:
HTML/CSS/JS
Forside, Meny- produktside (hardkodet), Bestillingsside og adminside for ansatte (begge statisk).

Teknologier: 
HTML, CSS, JavaScript
PostgreSQL eller MariaDB
Docker + Docker Compose
Markdown (README.md) for dokumentasjon
Git + Github
